# 一、【最推荐】用「自定义指令」当 System Prompt（永久生效）

这是**最稳、最省心**的方法，适合你这种**长期科研协作项目**。

## 适用效果

* 每一个新对话都会自动带着你的 **AP-POF 科研工作台人格**
* 不需要每次复制 prompt
* 非常适合长线论文项目

---

## 📍 操作步骤（桌面网页版 / Mac / Windows）

### 第 1 步：打开设置

1. 打开 ChatGPT 页面
2. 看 **左下角**（不是右上角）
3. 点击你的 **头像 / Initial 图标**
4. 点击 **Settings（设置）**

---

### 第 2 步：进入「自定义指令」

1. 在 Settings 左侧菜单中
2. 找到并点击
   **“Custom instructions”**（自定义指令）

> ⚠️ 如果你看到的是中文界面，名字可能是：
> **「自定义指令」**

---

### 第 3 步：你会看到两个输入框（这是关键）

#### 🔹 输入框 1（给 ChatGPT 的背景）

> **What would you like ChatGPT to know about you to provide better responses?**

👉 **这个框：可以留空，或者简单写一句**

```
I am conducting a long-term clinical machine learning research project based on ICU databases.
```

（不写也完全没问题）

---

#### 🔹 输入框 2（真正的 System Prompt）

> **How would you like ChatGPT to respond?**

👉 **把我刚刚给你的「ChatGPT System Prompt（完整版）」完整复制进去**

✔ 可以直接全贴
✔ 不用加引号
✔ 不用解释

---

### 第 4 步：保存

* 点击右下角 **Save**
* 完成 ✅

---

## ✅ 如何确认是否生效？

新建一个对话，然后直接输入一句：

```
我们开始 Module 00
```

如果 ChatGPT 回复类似：

> “我已锁定 AP-POF 研究流程，等待 Module 00 代码。”

说明 **100% 生效**。

# 二、【临时会话】用“第一条消息”模拟 System Prompt

如果你 **没找到自定义指令**，或在医院 / 公共电脑上用 ChatGPT，这是**备用方案**。

---

## 📍 操作方法

### 第 1 步：新建对话

点击 **New chat**

---

### 第 2 步：第一条消息直接粘贴

把我给你的这段内容 **整段粘贴并发送**：

> 🧠 ChatGPT System Prompt（完整版）
> ……（完整 prompt 内容）

---

### 第 3 步：等待确认

ChatGPT 回复确认后，再发：

```
Module 00-A 代码如下：
```

---

## ⚠️ 注意

* **只对当前对话有效**
* 关闭对话就失效
* 适合临时工作，不适合长期项目

# 三、【进阶】结合 Canvas（你已经在用）

你现在已经在用 Canvas（右侧流程文档），这其实是**最佳实践**。

# 四、常见失败原因（90% 的问题在这里）

### ❌ 1. 粘错输入框

很多人把 system prompt 粘到 **第一个框**
➡️ **一定要粘到第二个框**

---

### ❌ 2. 粘完没点 Save

切换页面会丢失
➡️ 一定点 **Save**

---

### ❌ 3. 老对话里测试

自定义指令 **不会 retroactively 生效**
➡️ 必须 **新建对话**

---

### ❌ 4. 期望“记住所有历史对话”

System prompt ≠ 记忆数据库
➡️ 超过 20 轮记得用你设计的 **Step 3 同步 Prompt**
